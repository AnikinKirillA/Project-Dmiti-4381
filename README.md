## Суть самого проекта:
В ходе проекта было реализованно 4 класса, Натуральные числа, Целые числа, Рациональные числа, а так же Полиномы. 
В каждом классе содержатся различные методы, для реализации которых использовались основные принципы ООП.
[Само задание](https://docs.google.com/document/d/1Dv_6AIhxg_3ezu6VMcEnMpyfRzgym9l8PmE4ULGfjgM/edit?usp=sharing)
Командная работа над проектом позволила в кротчайшие сроки качественно реализовать код, предоставило активное участие каждого члена комманды и представила наглядное примеенение теоретических знаний, полученных на лекциях.
Из-за чего каждый блок корректно выполняет свои функции, соответсвуя условию, и был протестирован, с учётом всевозможных пограничных случаев использывания.
